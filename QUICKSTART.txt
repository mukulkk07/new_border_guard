```bash

# 2. Follow prompts (enter GitHub username, token, repo name)

# 3. Activate environment
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows

# 4. Test installation
```

## Push to GitHub

```bash
# 1. Extract all 10 files (if not done)
# 2. Go to your local directory
cd github-automation

# 3. Create GitHub repo (online first)
# https://github.com/new

# 4. Initialize Git
git init
git add .
git commit -m "Add GitHub automation tools v1.0"
git remote add origin https://github.com/USERNAME/REPO.git
git branch -M main
git push -u origin main

# Done! Repository synced âœ“
```

## Get GitHub Token

1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Select these scopes:
   - âœ“ repo
   - âœ“ user (optional)
4. Copy token
5. Paste in setup.py prompt

**Token format:** `ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Module not found | `source venv/bin/activate && pip install -r requirements.txt` |
| .env missing | `python setup.py` |
| Git not found | Install Git from git-scm.com |
| pdflatex not found | Install TeX Live (for docs_pusher.py) |
| Permission denied | `chmod +x *.py` (Linux/macOS) |
